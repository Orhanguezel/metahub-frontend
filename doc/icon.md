# 🎯 Kullanım:

- Backend'de **Icon** alanına **bu tabloda gösterdiğim ikon adını** yazacaksın.
- Örneğin, `blog` için **MdArticle**, `order` için **MdReceiptLong** yazacaksın.
- Frontend otomatik dinamik `react-icons/md` içinden çekip gösterecek. ✅
